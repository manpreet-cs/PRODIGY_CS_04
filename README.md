## Simple Keylogger

### Task 04 – Keystroke Logging Tool

## Description
This project is a basic Python keylogger that records keyboard inputs and stores them in a text file. The project is created for educational and ethical cybersecurity learning purposes only.

## Features
- Records key presses
- Saves logs into a text file
- Stops safely using ESC key
- Simple command-line interface

## Technologies Used
- Python
- pynput library

## Requirements

Install pynput before running the program:

```bash
pip install pynput
```

## How to Run

```Bash
python keylogger.py
```

- Press keys on the keyboard.
- Press ESC key to stop the keylogger.

## Output
Logged keys are saved in:
``` Plain text
 keylog.txt
```

## Ethical Notice
This project is intended strictly for educational purposes and authorized environments only. Unauthorized use of keyloggers is unethical and may violate laws and privacy policies.
